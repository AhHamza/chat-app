# Chat App

A real-time chat application with user and room selection.

## Technologies
- Node.js, Express, JavaScript
- HTML, CSS
- Socket.io for real-time communication

## Key Features
- Users can enter a username and room name to join chats
- Send and receive messages in real-time
- Chat history maintained during the session

## What I Learned
- Implemented WebSockets using Socket.io for real-time communication
- Managed multiple chat rooms and handled user sessions in memory
- Improved understanding of asynchronous JavaScript and event-driven programming
